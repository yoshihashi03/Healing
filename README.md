# <Healing>

## サイト概要
一人でも、気軽に行ける、動物に癒される場所が見つかる・教えるサイト。

### サイトテーマ
日々の喧騒から抜け出し、癒されることが目的。

### テーマを選んだ理由
動物好きであるから。<br>
ファミリー向けが行きにくい人でも、気軽に行ける場所を見つけられるよう。

### ターゲットユーザ
お一人様、学生、カップルなど。

### 主な利用シーン
休日・仕事(学校)終わり・休憩時間など。
